# ZenetHunter

> 소규모 네트워크와 홈 LAN 환경을 위한 장치 가시성, 관측, 접근 제어, 로컬 방어 오케스트레이션 저장소입니다.

> 안전 안내: 이 저장소에는 네트워크 스캔 및 능동 방어 관련 코드가 포함되어 있습니다. 반드시 본인이 소유하거나 명시적으로 승인을 받은 네트워크에서만 사용하십시오.

---

## AI 에이전트 지침

AI 에이전트는 이 저장소를 수정하기 전에 반드시 `AGENT/`를 먼저 읽어야 합니다. 지정된 읽기 순서를 완료하기 전에는 `AGENT/` 밖의 코드, 계약, 문서를 수정하면 안 됩니다.

필수 읽기 순서:
1. `AGENT/memory/project-purpose.md`
2. `AGENT/rules/safety-context.md`
3. `AGENT/bridge/*`
4. `AGENT/content/*`
5. `AGENT/tasks/current-branch-plan.md`

`AGENT/`에는 이 저장소 전용 프로젝트 메모리, 안전 경계, 프런트엔드/백엔드 브리지 계약, 저장소 맵, 현재 브랜치 계획이 기록됩니다. 이 안전 맥락은 이 저장소에만 적용되며, 재사용 가능한 일반적 면제 패턴으로 일반화하면 안 됩니다.
