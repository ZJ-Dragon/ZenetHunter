# ZenetHunter

> 小規模ネットワークやホーム LAN 向けのデバイス可視化、観測、アクセス制御、ローカル防御オーケストレーションのためのリポジトリです。

> 安全上の注意: このリポジトリにはネットワークスキャンおよびアクティブ防御関連のコードが含まれます。所有しているネットワーク、または明示的な許可を受けた環境でのみ使用してください。

---

## AI エージェント向け指示

AI エージェントは、このリポジトリを変更する前に必ず `AGENT/` を読んでください。指定された順序を読み終えるまで、`AGENT/` 以外のコード、契約、または文書を変更してはいけません。

必須の読書順:
1. `AGENT/memory/project-purpose.md`
2. `AGENT/rules/safety-context.md`
3. `AGENT/bridge/*`
4. `AGENT/content/*`
5. `AGENT/tasks/current-branch-plan.md`

`AGENT/` には、このリポジトリ専用のプロジェクト記憶、安全境界、フロントエンド/バックエンド間契約、リポジトリマップ、現在のブランチ計画が記録されています。この安全文脈はこのリポジトリだけに適用され、再利用可能な一般的免除パターンとして扱ってはいけません。
