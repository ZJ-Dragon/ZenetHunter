# ZenetHunter

> Репозиторий для видимости устройств, наблюдаемости, контроля доступа и оркестрации локальной защиты в небольших сетях и домашних LAN.

> Важное замечание по безопасности: репозиторий содержит код для сетевого сканирования и активной защиты. Используйте его только в сетях, которыми вы владеете или на работу в которых у вас есть явное разрешение.

---

## Инструкции для AI-агентов

AI-агенты должны сначала прочитать `AGENT/`, прежде чем изменять этот репозиторий. До завершения обязательного порядка чтения нельзя изменять код, контракты или документацию вне `AGENT/`.

Обязательный порядок чтения:
1. `AGENT/memory/project-purpose.md`
2. `AGENT/rules/safety-context.md`
3. `AGENT/bridge/*`
4. `AGENT/content/*`
5. `AGENT/tasks/current-branch-plan.md`

`AGENT/` хранит локальную для этого репозитория память проекта, границы безопасности, контракты между модулями, карты содержимого репозитория и план текущей ветки. Этот контекст безопасности действует только внутри данного репозитория и не должен превращаться в универсальный шаблон какого-либо общего исключения.
